# Magma — Archin Studio Website

A modern architectural studio website built with ReactJS, SCSS Modules, GSAP animations, and SwiperJS. Recreated from the [Archin ReactJS template](https://archin-reactjs.vercel.app/home1).

## 🚀 Live Demo

[View on GitHub Pages](https://junaid212.github.io/magma)

## 🛠 Tech Stack

- **ReactJS** (JavaScript)
- **Bootstrap 5**
- **SCSS Modules**
- **GSAP + ScrollTrigger** — scroll animations
- **Lenis** — smooth scrolling
- **SwiperJS** — image sliders
- **React Router DOM** — client-side routing
- **React Icons**

## 📦 Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm start

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```

## 📁 Project Structure

```
src/
├── components/       # Shared components (Navbar, Footer, Loader, etc.)
├── pages/
│   └── Home/
│       └── sections/ # All home page sections
├── styles/           # Global SCSS variables & mixins
└── utils/            # GSAP config, helpers
```
